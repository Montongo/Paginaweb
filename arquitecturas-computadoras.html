Arquitecturas de Computadoras

Explora las diferentes arquitecturas de computadoras, sus tipos, características y funcionamiento.

Arquitectura Von Neumann

La arquitectura Von Neumann es una de las más tradicionales y se basa en una única memoria principal que almacena tanto instrucciones como datos. La CPU accede a esta memoria a través de un sistema de buses único.

  • Características: Memoria única para datos e instrucciones, buses compartidos.
  • Funcionamiento: La CPU ejecuta instrucciones secuencialmente, accediendo a la memoria para obtener datos y comandos.

Arquitectura Harvard

La arquitectura Harvard utiliza vías separadas para instrucciones y datos, lo que permite un acceso más rápido y eficiente. Es común en microcontroladores y procesadores de señal digital.

  • Características: Memorias separadas para datos e instrucciones, buses independientes.
  • Funcionamiento: Permite la ejecución simultánea de operaciones de lectura de instrucciones y acceso a datos.

Arquitectura ARM

ARM es una arquitectura RISC de 32 y 64 bits, conocida por su eficiencia energética y amplio uso en dispositivos móviles y sistemas embebidos.

  • Características: Conjunto de instrucciones reducido, bajo consumo de energía.
  • Funcionamiento: Ejecuta instrucciones simples en un solo ciclo de reloj, optimizando el rendimiento y la eficiencia.

Componentes de la CPU

La CPU está compuesta por varios componentes clave que trabajan juntos para ejecutar instrucciones y procesar datos.

  • ALU (Unidad Aritmético-Lógica): Realiza operaciones aritméticas y lógicas.
  • Unidad de Control: Coordina las actividades de la CPU, gestionando el flujo de datos y ejecución de instrucciones.
  • Registros: Almacenan datos temporalmente durante la ejecución de instrucciones.
  • Buses Internos: Facilitan la comunicación entre los componentes de la CPU y la memoria.
